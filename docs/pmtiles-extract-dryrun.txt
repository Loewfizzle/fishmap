**Computed bbox source:** `data/aoi.geojson` (exact 40-mile polygon bounds, see scripts/aoi.py and DESIGN.md).

**Rationale for --maxzoom=12 (not 13 or 14):**
- Per DESIGN.md "PMTiles Sizing & Optimization Notes": z13 roughly doubles size vs z12 in vector data.
- For mobile 4G practicality (sub-3s initial + full region download target <10min on typical connection), z12 is the conservative default for the outer AOI radius.
- z13 reserved for dense urban cores (Grand Rapids + immediate Kent); full PR6 dataset will document per-layer zoom stats.
- Dry-run confirms feasibility before committing any regional extract (we never commit the ~GB-scale full-planet slice; only use for hosted R2 per ASSET-HOSTING.md decision).
